Blockchain Technology Education Tool

Languages Used: Java

IDE/Applications Used: Eclipse, XML

  • Java project that simulated blockchain technology with 3 nodes.
  • Uses script to start three different processes of Blockchain nodes where they each compete to verify a puzzle.
  • A Ledger is updated with node that verified puzzle first and the nodes communicate with each other to know who solved the puzzle and when to stop.
  • Operated through the command line and creates an output XML file that shows how the final result and how the nodes communicate with each other.
  • Simulates how Blockchain/Ledger-based entities that large companies may use, operate in a smaller and easier to interpret setting.
  • Screenshot shows the output of one process communicatin with the other nodes about a puzzle being solveed and updating the ledger.

REST API Quote Service

Languages Used: Java

IDE/Applications Used: Eclipse, Maven, XML

  • REST API Service where user connects to service and can request and make changes through through HTTP requests.
  • Retrieves specific quotes or all quotes via GET HTTP request with specific URI.
  • Functionality to display all quotes via index pagination where you can specify the amount of pages and quotes per page with specific URI.
  • Can update quotes with a PUT HTTP request and can add a new quote with POST HTTP request.
  • Has funtionality to delete a quote using a DELETE HTTP request.
  • Can make requests through the command line using curl command and through web browser.
  • Overall, illustrates functionality of how a REST service interprets a request and sends back the requested information back to the client.
